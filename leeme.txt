# Convertidor de Imagen a ASCII

Este es un convertidor de imágenes a ASCII que convierte imágenes en formato JPG/PNG a representaciones ASCII utilizando el módulo Python PIL y la biblioteca web Flask.

## Características

- Permite seleccionar y cargar una imagen desde un formulario web.
- Convierte la imagen a una representación ASCII.
- Muestra la representación ASCII de la imagen en la página web.

## Requisitos previos

- Python 3.x instalado en tu sistema.
- Las dependencias se encuentran en el archivo `requirements.txt` y se pueden instalar ejecutando `pip install -r requirements.txt`.

## Personalización

- Puedes personalizar el tamaño de la imagen resultante modificando el valor de `new_width` en el archivo `app.py`.

- Puedes personalizar el diseño y la apariencia de la página HTML en el archivo `index.html`.

## Limitaciones

- La aplicación actualmente solo admite imágenes en formato JPG y PNG.

- La aplicación no implementa validación exhaustiva de archivos de imagen. Asegúrate de cargar solo imágenes válidas.
