Conversor de Imágenes a ASCII
Este es un pequeño proyecto que convierte imágenes en ASCII utilizando Flask y la biblioteca PIL.

Descripción
El conversor de imágenes a ASCII es una aplicación web que permite cargar imágenes y convertirlas en representaciones ASCII. El proceso implica redimensionar la imagen, convertirla a escala de grises y mapear los valores de brillo de los píxeles a caracteres ASCII predefinidos.

Requisitos
Asegúrate de tener instalado lo siguiente antes de ejecutar la aplicación:

Python 3
Flask
PIL (Pillow)
Uso
Clona el repositorio o descarga los archivos en tu máquina.

Instala las dependencias mencionadas anteriormente.

Ejecuta la aplicación utilizando el siguiente comando:
python app.py
Abre tu navegador web y visita http://localhost:5000 para acceder a la aplicación.

En la página principal, puedes seleccionar un archivo de imagen desde tu dispositivo o proporcionar una URL de imagen.

Especifica el tamaño deseado para la imagen ASCII y haz clic en el botón "Convertir".

La imagen convertida se mostrará en la misma página.
